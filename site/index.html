Numbers, not adjectives

One 0.5B model, five arms, identical execution-based grading: every generated function is run against generated test cases and checked against contracts. The Claw arm had a ~$0.03 fine-tune on machine-generated data. The other arms had the model's entire pretraining corpus.

Four gates, one guarantee

No single layer makes a model reliable. The pipeline is the product.

gate 1

Code as database

Every definition is content-addressed and queryable. An agent asks what exists with this type? instead of guessing an API. Measured: hallucinated symbols 38 → 0 from context alone.

gate 2

Constrained decoding

The scope projects to a grammar. Out-of-scope calls and undeclared effects are ungeneratable at the token level — a guarantee, not a prompt.

gate 3

Real typecheck

Generated code round-trips through the actual compiler. Wrong arity, undefined helpers, ignored parameters — caught before a human ever reads the diff.

gate 4

Executable contracts

Definitions carry postconditions that run on generated inputs. area(2,3) typechecks; result == w*h doesn't lie.

Sixty seconds in

# fizzbuzz.claw
fizz : U64 -> Str
fizz = |n| {
    if n % 15 == 0 "FizzBuzz"
    else if n % 3 == 0 "Fizz"
    else if n % 5 == 0 "Buzz"
    else n.to_str()
}

main! = |_args| {
    echo!(fizz(15))
    Ok({})
}
$ claw run fizzbuzz.claw
FizzBuzz
