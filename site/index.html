A language and a proof

Built for AI to write.

Claw is a fast, functional language where code lives in a queryable database, generation is grammar-constrained, and every definition is verified by a real compiler and executable contracts — before a human reads the diff.

Why Claw

Every language claims safety. Claw's claim is narrower and checkable: an AI writing Claw cannot reference what doesn't exist, cannot skip declaring its effects, and cannot ship code that fails its contracts.

Verifiable by construction

Four independent gates — a code database that answers "what exists?", a decode grammar that makes out-of-scope calls ungeneratable, the real compiler's typecheck, and contracts that execute. No single gate is trusted; the pipeline is.

AI-native, not AI-flavored

The database, the grammar, and the verifier are APIs — an MCP server any agent can call today, from Claude Code to Cursor to Gemini CLI. A bundled fine-tuned model ships with the toolchain, and opt-in telemetry makes the next one better.

Small, fast, complete

A functional language with effect types, pattern matching, and 100–200 ms typechecks. Compiler, formatter, test runner, LSP, MCP server, and self-updates in one tarball. No system dependencies.

What it looks like

Real programs from the repository — click through.

fizz : U64 -> Str
fizz = |n| {
    if n % 15 == 0 "FizzBuzz"
    else if n % 3 == 0 "Fizz"
    else if n % 5 == 0 "Buzz"
    else n.to_str()
}

main! = |_args| {
    echo!(fizz(15))
    Ok({})
}
# Tags and match — how Claw handles absence and failure.
classify = |n| match n {
    0 => "zero"
    1 => "one"
    _ => "many"
}

main! = |_args| {
    labels = [0, 1, 2, 3].map(classify)
    echo!(Str.join_with(labels, ", "))
    Ok({})
}
# Effects are visible in the name (!) and checked in the type.
# A pure function cannot quietly start reading files.
cat_file! : Str => Str
cat_file! = |path| {
    contents = File.read!(path)
    Stdout.line!(contents)
    contents
}
# The code IS a database. Agents query it instead of guessing.
$ claw db candidates "Str -> Str"
Str.upper  : Str -> Str
File.read! : Str -> Str  [effects: Fs]

$ claw db callers Nat.add        # impact analysis, O(1)
$ claw db rename old new          # no caller ever breaks
$ claw task-grammar task.json     # the decode grammar for this scope
