A new programming language, with proof

AI writes the code.
Claw makes sure it's real.

AI assistants often invent functions that don't exist — and you find out after the bug ships. Claw is a programming language that checks every line an AI writes against what actually exists in your project, typechecks it with a real compiler, and runs the tests — before you ever read it.

Get started Try the playground
$ curl -fsSL https://clawlang.dev/install.sh | sh
# Your AI assistant just invented a function.
$ claw defs-check clamp.json task.json
COMPILE-FAIL (1 error)
  `Nat.clamp` does not exist in this
  project. Caught before you saw it.

# The AI asks what really exists instead:
$ claw db candidates "Nat, Nat -> Nat"
Nat.max : Nat, Nat -> Nat
Nat.min : Nat, Nat -> Nat

✓ min(max(x, lo), hi) — verified, 2/2 contracts hold

Why Claw

Every language promises safety. Claw's promise is simpler, and you can verify it yourself: an AI writing Claw can't call things that don't exist, can't secretly touch your files or network, and can't ship code that fails its own tests.

Four safety checks, zero trust

A live map of your project answers "what exists?". A grammar makes made-up names literally impossible to generate. The compiler checks the types. The tests actually run. No single check is trusted — all four are.

Works with your AI today

Claude Code, Cursor, Windsurf, VS Code, Zed, Gemini CLI, Codex — one config block connects any of them to your project's live code map. Claw even ships its own small AI model, trained to write it correctly.

Everything in one download

Compiler, formatter, test runner, editor support, and AI integration in a single install — one command, no dependencies, updates itself. Checks your code in about a tenth of a second.

What it looks like

Real programs from the repository — click through.

fizz : U64 -> Str
fizz = |n| {
    if n % 15 == 0 "FizzBuzz"
    else if n % 3 == 0 "Fizz"
    else if n % 5 == 0 "Buzz"
    else n.to_str()
}

main! = |_args| {
    echo!(fizz(15))
    Ok({})
}
# Tags and match — how Claw handles absence and failure.
classify = |n| match n {
    0 => "zero"
    1 => "one"
    _ => "many"
}

main! = |_args| {
    labels = [0, 1, 2, 3].map(classify)
    echo!(Str.join_with(labels, ", "))
    Ok({})
}
# Effects are visible in the name (!) and checked in the type.
# A pure function cannot quietly start reading files.
cat_file! : Str => Str
cat_file! = |path| {
    contents = File.read!(path)
    Stdout.line!(contents)
    contents
}
# The code IS a database. Agents query it instead of guessing.
$ claw db candidates "Str -> Str"
Str.upper  : Str -> Str
File.read! : Str -> Str  [effects: Fs]

$ claw db callers Nat.add        # impact analysis, O(1)
$ claw db rename old new          # no caller ever breaks
$ claw task-grammar task.json     # the decode grammar for this scope

Questions people ask

What is Claw, in one sentence?

A free, open-source programming language where code written by AI is checked against reality — so invented functions and silent mistakes never reach you.

Why does AI need its own language?

AI assistants confidently call functions that don't exist. Existing languages can't stop that; Claw was designed so it's caught — or can't be generated at all. Measured: 38 invented calls per test run → 0.

Do I need to know AI or machine learning?

No. Claw is a normal language — install, write, run. The AI features work when you want them and stay out of the way when you don't.

Does it work with Claude, Cursor, Copilot?

Yes — one config block connects Claude Code & Desktop, Cursor, Windsurf, VS Code, Zed, Gemini CLI, or Codex to your project's code map.

Is it really free?

Yes. UPL-1.0 license — compiler, tools, AI model, everything. No tracking on this site. The toolchain collects anonymous usage metrics (never your code) — claw telemetry off disables it; sharing code samples for training is strictly opt-in.

Who makes Claw?

LambdaQ Labs. The language builds on the open-source Roc compiler; the AI layer, benchmark, and tooling are developed in the open on GitHub.
