# Baseline Run — 2026-07-03

First real gate data. Model: **codestral-latest** (Mistral API), 15 tasks, ≤2 retries, temperature 0.2. Raw JSON: `bench/results/A{0,1}-codestral.json` (not committed; regenerate with the commands below).

## Results

| Metric | A0 (blind) | A1 (+CDB symbol table) | Δ |
|---|---|---|---|
| Compiled (no dangling refs) | 0/15 | **10/15** | +67pts |
| Tasks with hallucinations | 10/15 | **1/15** | −90% |
| Hallucinated symbols total | 28 | **1** | **−96%** |

- A0 hallucinations were the classic API-invention class: `add`, `fold`, `++`, `split`, `head`, `filter`, `length`, … — symbols that plausibly exist in most stdlibs but are not in scope.
- The single A1 "hallucination" (`retryWithBackoff`) is a self-recursive reference — the Def-JSON output protocol has no way to name a def yet, so self-reference is unrepresentable. Arguably a true 0. **Protocol TODO: named defs.**
- `passed = 0/15` everywhere **by design**: test/contract oracles don't execute until the compiler-integrated test runner lands; tasks with unexecuted oracles never silently pass (harness spec §7). Compile + hallucination are the live metrics.

## Interpretation

The A0→A1 delta measures exactly one intervention: **including the code-as-database symbol table (names + types) in the prompt.** That single change eliminated 96% of symbol hallucination and took compile-shaped success from 0 to 2/3. This reproduces, on our own harness, the published finding that API hallucination is the dominant failure class (41.3% of compile failures in LLM-generated Rust crypto code, arXiv:2604.27001).

P2's formal gate compares **A2** (decode-time grammar mask) vs A0 with a >30% compile-error-reduction bar. A1's −96% hallucination already exceeds that bar's spirit; A2 should structurally guarantee ~0.

## Gaps / next

- **A2 run needs a grammar-honoring server** (llama.cpp `--grammar` / `grammar` request field). Mistral's API silently ignores the field, so an A2 run there would just duplicate A1. Do not report such a run as A2.
- Self-reference: extend the output protocol with named defs so recursive functions are expressible; re-run.
- Ollama on this machine (M1, 8 GB) has only gemma4:e4b (9.6 GB — would swap); a ≤3B coder pull is the local path if needed.
